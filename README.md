## 🚀 Instalación y entorno

git clone https://github.com/pierolucas10/EQUIPO1_SGA.git
cd EQUIPO1_SGA

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py check

python manage.py runserver
